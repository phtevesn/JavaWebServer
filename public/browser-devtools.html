Browser DevTools for HTTP

Modern browsers have powerful developer tools for inspecting HTTP traffic. Press F12 to open DevTools in most browsers.

Opening the Network Tab

  1. Press F12 (or Ctrl+Shift+I / Cmd+Option+I)
  2. Click the Network tab
  3. Refresh the page or make requests
  4. Click on any request to see details

What You Can See

Request Headers

View all headers your browser sent:

  • Host, User-Agent, Accept headers
  • Connection header (keep-alive or close)
  • Custom headers if any

Response Headers

View all headers the server sent:

  • Status code (200, 404, etc.)
  • Content-Type, Content-Length
  • Connection header
  • Custom headers like X-Payload-Hash

Timing Breakdown

See exactly where time is spent:

  • DNS Lookup: Time to resolve hostname
  • Connecting: TCP handshake time
  • Waiting (TTFB): Time to first byte
  • Content Download: Time to download response

Testing the Echo Endpoint

  1. Open DevTools Network tab
  2. Click one of the echo test links on the home page
  3. In Network tab, click the /echo/... request
  4. View headers, timing, and the X-Payload-Hash!

Observing Keep-Alive

Look for the Connection ID column in Chrome DevTools:

  • Same Connection ID = requests reused same TCP connection
  • Different Connection ID = new TCP connection for each

Useful Keyboard Shortcuts

  • F12: Open/close DevTools
  • Ctrl+R: Refresh page
  • Ctrl+Shift+R: Hard refresh (bypass cache)
  • Escape: Toggle console drawer
